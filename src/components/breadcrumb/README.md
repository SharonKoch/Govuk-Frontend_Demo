## Arguments

| Name        | Type   | Default | Required | Description
|---          |---     |---      |---       |---
| breadcrumbs | array  |         | Yes      | Breadcrumbs array with title and url keys
| title       | string |         | Yes      | Title of the breadcrumb item
| url         | string |         | Yes      | Url of the breadcrumb item
